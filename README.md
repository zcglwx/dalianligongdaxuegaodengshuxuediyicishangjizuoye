2. **微分计算**
   - 定义符号变量 `t` 和 `a`。

   - 计算 `y` 对 `x` 的一阶导数 `yx`，结果为 `-sin(t)/(cos(t) - 1)`。
   - 计算 `y` 对 `x` 的二阶导数 `yxx`，结果为 `(sin(t)^2`。
